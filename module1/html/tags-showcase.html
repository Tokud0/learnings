HTML — витрина тегов (Модуль 1)

Здесь много маленьких примеров. Смотрите: как выглядит в браузере и как это пишется в коде.

1) Семантическая структура страницы

Эти теги помогают читать страницу людям и поисковикам (и помогают со стилями).

header: шапка
nav: меню

section: секция

article: статья/пост

Текст статьи...

aside: боковая колонка
footer: подвал

Код:

<header>...</header>
<nav>...</nav>
<main>
  <section>
    <article>...</article>
  </section>
  <aside>...</aside>
</main>
<footer>...</footer>

8) Общие атрибуты (важные)

  • id — уникальный идентификатор (для CSS/JS/label-for).
  • class — классы для CSS/JS (можно несколько через пробел).
  • style — инлайн-стиль (быстро, но лучше выносить в CSS).
  • title — подсказка при наведении.
  • hidden — скрыть элемент.
  • data-* — свои данные (например, data-id="123").
  • aria-* — доступность (например, aria-label).
