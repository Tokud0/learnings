# Внеурочный курс: Модуль 1 (HTML/CSS/JS/Forms)

## Точка входа
- `module1/html/index.html` — главная навигация по всем демкам.

## HTML
- `module1/html/tags-showcase.html` — витрина HTML‑тегов и атрибутов; формы тут только скелет, подробности вынесены отдельно.
- `module1/html/forms-httpbin.html` — примеры форм: GET, POST, POST через fetch; отправка на https://httpbin.org (эхо‑сервер) с выводом ответа.

## CSS
- `module1/html/css-showcase.html` — наглядные примеры свойств: размеры, цвет/фон, текст, margin vs padding, border/radius, тени/hover/transition, таблицы (границы, zebra, hover), селекторы.
- `module1/css/css-showcase.css` — стили для CSS-витрины (переменные, утилиты, демо-классы, топ‑навигация).

## JavaScript
- `module1/html/js-basics-cheatsheet.html` — простая “читаемая” база (открыть DevTools → Console): переменные, условия, циклы, массивы, функции, объекты/JSON.
- `module1/js/js-basics.js` — код для шпаргалки (все примеры через console.log, с комментариями что получится).
- `module1/html/js-basics.html` — интерактивная база: слева показывается код фрагмента, справа вывод; кнопки по разделам.
- `module1/js/js-basics-demo.js` — логика интерактивной базы (примеры + вывод на страницу и в console).
- `module1/html/js-showcase.html` — мини‑витрина DOM/событий/classList/dataset (кнопка, переключение темы, чтение data‑атрибутов).
- `module1/js/js-showcase.js` — код витрины DOM.

## Прочее
- `module1/js/forms-httpbin.js` — JS для отправки форм на httpbin (пример fetch без перезагрузки).

## Как открыть
1. Запустите `module1/html/index.html` в браузере (двойной клик или “Open in browser”).
2. Переходите по ссылкам в шапке или по списку на странице.

## Полезно знать
- Логи в JS примерах: DevTools → Console (F12).
- Интерактивная база (`js-basics.html`): код слева, результат справа, плюс вывод в консоль.
- Формы: отправка на `https://httpbin.org`, смотрите вкладку Network → Payload/Response.
